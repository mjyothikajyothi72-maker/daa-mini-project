# 🏥 Hospital Priority System using Heap Sort

## 📌 Description
This project implements a hospital priority system using the **Heap Sort algorithm (Max Heap)**. Patients are assigned priority values, and the system sorts them in **descending order**, ensuring that higher-priority patients are treated first.

## ⚙️ Features
- Accepts patient priority values  
- Uses Heap Sort (Max Heap)  
- Displays patients in descending priority order  
- Simple and efficient C program  

## 💻 Technology Used
- Language: C Programming  
- Concept: Data Structures (Heap Sort)
